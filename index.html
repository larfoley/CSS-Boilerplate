Getting Started

Include the following in the head section of you HTML document.

<link rel="stylesheet" href="/css/master.css">

To download the src code type the following command.

git clone https://github.com/larfoley/boilerplate/

Buttons

Button Skins

<button class="btn">Default Button</button>
<button class="btn btn-primary">Primary Button</button>
<button class="btn btn-success">Success Button</button>
<button class="btn btn-warning">Warning Button</button>

Colors

Adjust the background color, text-color and border color of any HTML elemnet.

  • Background
  • Border
  • Colored Text
<div class="red">Background</div>
<div class="blue-border">Border</div>
<div class="green-text">Colored Text</div>

Available Color Classes:

  • red
  • green
  • blue
  • brightgray
  • radical-red

Container

A container centers blocks of content.

This text is inside a container

<div class="container">This text is inside a container</div>

Section

Sections space out the top and bottom padding of an element.

this is a section

<div class="section">This text is inside a section</div>

Box

Boxes have padding applied to the top, right, bottom and left of an element.

I'm a box

<div class="box">i'm a box</div>

Utilities

Helpfull utilite classes. Note that all utilite classes use the !important rule.

  .u-text-center {
    text-align: center !important;
  }

  .u-inline {
    display: inline-block !important;
  }

  .u-block {
    display: block !important;
  }

  .u-padding {
    padding: 1em !important;
  }

  .u-margin {
    margin: 1em !important;
  }
